
# Lecture Voice-to-Notes Generator

An AI-powered system that converts lecture audio into text 
and generates summarized study notes.

## Features
- Speech-to-Text Conversion
- AI Text Summarization
- Simple Web Interface (Streamlit)

## Technologies Used
- Python
- SpeechRecognition
- HuggingFace Transformers
- Streamlit

## Deployment
Deploy using Streamlit Cloud.
